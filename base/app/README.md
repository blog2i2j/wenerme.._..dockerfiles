## wener/base:openrc

在 Docker 中运行 openrc 服务, 默认启用了 syslog 和 local 服务。创建了 admin 账号，但禁用了密码

- syslog
  - 用于服务日志记录 - 例如 ssh
- local
  - 用于启动本地脚本
